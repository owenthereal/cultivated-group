# CultivatED Group

Company website for CultivatED Group — ECE consulting services and platform.

**Live:** [cultivatedgroup.ca](https://cultivatedgroup.ca) (Cloudflare Pages)

## Development

```bash
npm install          # Install dependencies
npm run dev          # Start dev server (localhost:4321)
npm run build        # Build for production
npm run deploy       # Build and deploy to Cloudflare Pages
```

## Tech Stack

- [Astro](https://astro.build/) — static site generator (SSR when backend is needed)
- [Tailwind CSS](https://tailwindcss.com/) — utility-first styling
- [Cloudflare Pages](https://pages.cloudflare.com/) — hosting and deployment
